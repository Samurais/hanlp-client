# hanlp client
## Welcome
```
npm install hanlp-client
```

## Configure
By default, **hanlp-client** connected with [nlp.chabot.io](http://nlp.chatbot.io/public/index.html).

Just set **HANLP\_SERVER\_URL** as an environment variable for customized URL.
